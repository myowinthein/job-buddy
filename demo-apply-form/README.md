# Job Buddy Demo Application Page

A fictional job application page for testing and marketing screenshots for the Job Buddy Chrome extension.

This page is original and only inspired by common ATS layouts. It does not use third-party logos, branding, assets, CSS, or text.

## Files

- `index.html`
- `assets/css/styles.css`
- `assets/js/main.js`

## Usage

Open `index.html` in a browser.

For screenshot/demo convenience:

- Press `F` to simulate Job Buddy-style autofill highlights.
- Click `Clear form` to reset the page.
- Or run these in the browser console:
  - `JobBuddyDemo.fill()`
  - `JobBuddyDemo.clear()`

## Notes

The page intentionally includes common job application fields:

- Personal information
- Address
- Professional details
- Work authorization
- Experience
- Education
- Resume upload

Use this as a safe demo page for Chrome Web Store screenshots, README images, and product demos.
